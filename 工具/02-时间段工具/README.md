# 时间段工具说明

本目录存放时间段生成相关的工具脚本。

## 📄 工具列表

### generate_timeslots.py
- **功能**：生成标准的上课时间段数据
- **用途**：初始化时间段数据、批量创建时间段
- **使用方法**：
  ```bash
  python 工具/02-时间段工具/generate_timeslots.py
  ```
- **Windows快速运行**：双击 `generate_timeslots.bat`

### generate_timeslots.bat
- **功能**：Windows批处理文件，快速运行时间段生成工具
- **使用方法**：双击运行即可

## ⚠️ 注意事项

1. **数据库连接**：确保数据库配置正确
2. **权限**：确保有写入数据库的权限
3. **重复检查**：脚本会自动检查时间段是否已存在，不会重复创建
4. **数据安全**：此工具会创建数据，运行前建议备份数据库

## 💡 使用场景

- **初始化系统**：首次部署时生成时间段数据
- **补充时间段**：添加缺失的时间段
- **重置时间段**：清空后重新生成标准时间段

## 🔄 运行结果

运行后会显示：
- 新创建的时间段数量
- 已存在的时间段数量

- 时间段概览信息

## ❓ 常见问题

**Q: 时间段已存在怎么办？**
A: 脚本会自动跳过已存在的时间段，不会重复创建。

**Q: 可以修改时间段吗？**
A: 可以修改脚本中的时间配置，然后重新运行。注意：已存在的时间段不会被更新，需要手动修改数据库。

**Q: 如何删除所有时间段？**
A: 可以通过Django管理后台或数据库直接删除，然后重新运行此脚本生成。
