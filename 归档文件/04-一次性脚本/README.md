## 📄 文件列表

### update_class_ids.py
- **原位置**：`organization/management/commands/update_class_ids.py`
- **用途**：批量更新所有班级的class_id字段
- **功能**：
  - 根据年级、学制、学院代码、专业代码、班级序号生成新的class_id
  - 格式：年级ID(4位) + 学制数(1位) + 学院ID(2位) + 专业ID(2位) + 班级序号(1位)
  - 例如：2024301011 = 2024年 + 3年制 + 01学院 + 01专业 + 1班
- **任务状态**：✅ 已完成
- **是否可以删除**：✅ 是，任务已完成，不再需要
- **使用方法**（仅供参考）：
  ```bash
  python manage.py update_class_ids
  ```
