# EduCloud Design System & Style Guide

## 1. Design Philosophy
The EduCloud interface follows a **Modern Dark** aesthetic inspired by Fluent Design and Material Design 3.
- **Core Concept**: Depth, Light, and Motion.
- **Backgrounds**: Deep grays (#202020) to reduce eye strain.
- **Surfaces**: Layered cards with subtle borders and shadows to establish hierarchy.
- **Accent**: A vibrant Sky Blue (#60cdff) to guide user attention.

## 2. Color Palette

| Token | Value | Usage |
|-------|-------|-------|
| `--bg-body` | `#202020` | Main background |
| `--bg-card` | `#2d2d2d` | Card / Sidebar background |
| `--bg-card-hover` | `#363636` | Hover states |
| `--text-primary` | `#ffffff` | Headings, Body text |
| `--text-secondary` | `#b3b3b3` | Subtitles, Metadata |
| `--accent-color` | `#60cdff` | Primary buttons, Active states |
| `--border-color` | `#404040` | Dividers, Borders |

## 3. Typography
- **Font Family**: 'Segoe UI', 'Roboto', 'Helvetica Neue', sans-serif.
- **Headings**: Bold, High contrast.
- **Body**: Regular, High legibility.

## 4. Components

### Buttons
- **Primary**: Solid accent color, black text for contrast.
- **Secondary/Outline**: Transparent background, border color matches text.

### Cards
- **Style**: Rounded corners (12px), subtle border (1px solid #404040).
- **Interaction**: Slight lift or glow on hover is optional but recommended for interactive cards.

### Navigation (Sidebar)
- **Layout**: Fixed width (260px) on desktop, collapsible on mobile.
- **Active State**: Highlighted background with left accent border.

### Data Visualization (ECharts)
- **Theme**: Dark mode compatible.
- **Colors**: Gradient fills for area charts, distinct colors for categories.
- **Interactivity**: Tooltips enabled, smooth transitions.

## 5. Grid & Layout
- **System**: Bootstrap 5 Grid System (Row/Col).
- **Spacing**: 
  - Small: 8px (`--spacing-sm`)
  - Medium: 16px (`--spacing-md`)
  - Large: 24px (`--spacing-lg`)

## 6. Responsive Behavior
- **Mobile (<768px)**: Sidebar hides off-canvas, toggled via hamburger menu.
- **Desktop**: Sidebar persistent.
